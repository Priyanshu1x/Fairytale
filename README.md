# A Fairytale 🎬

A cinematic Netflix-style movie intro landing page — built as a personal birthday surprise.

## Features

- 🎥 Netflix intro video with synced audio
- ✨ Cinematic hero section with Ken Burns zoom & parallax
- 🌟 Floating golden dust particles
- 🎭 Film grain & vignette overlays
- ▶️ Play button with ripple glow animation
- 📜 "Trending Memories" horizontal scroll row
- 🔊 Background music support
- 📱 Fully responsive (desktop + mobile)
- 🎨 Glassmorphism & premium dark UI

## Tech Stack

Pure **HTML**, **CSS**, and **JavaScript** — no frameworks.

## Setup

1. Clone the repo
2. Replace placeholder images in `assets/` with your own photos
3. Add a `music.mp3` to `assets/` for background music
4. Open `index.html` in a browser (or use a local server)

## File Structure

```
├── index.html          # Main landing page
├── birthday.html       # Destination after Play
├── style.css           # Design system & animations
├── script.js           # Interactivity & effects
└── assets/
    ├── netflix-intro.mp4
    ├── netflix-intro.mp3
    ├── netflix-logo.png
    ├── hero-bg.png
    ├── card1-6.png
    └── music.mp3 (optional)
```
